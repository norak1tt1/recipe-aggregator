# Установка и запуск

## 1. Скачать проект

**Способ 1:**
- На странице репозитория нажмите зелёную кнопку **"Code"**
- Выберите **"Download ZIP"**
- Распакуйте архив в любую папку

**Способ 2:**
```bash
git clone https://github.com/norak1tt1/recipe-aggregator.git
cd recipe-aggregator
```

---

## 2. Установить зависимости

Откройте командную строку в папке с проектом и выполните:

```bash
npm install
```

---

## 3. Запустить сервер

```bash
node server/index.js
```

---

## 4. Открыть в браузере

Перейдите по адресу:

```
http://localhost:3000
```

---
